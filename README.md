# ☄️ ANTIGRAVITY — Futuristic Gaming Landing Page

A visually immersive futuristic landing page designed for a fictional multiplayer sci-fi shooter game called **ANTIGRAVITY: Cosmic Warfare**.
Built using modern frontend technologies with responsive layouts, glowing UI effects, and interactive gaming aesthetics.

---

## 🚀 Live Demo

https://mr-utkarsh-git.github.io/landing-page/

---

## 📌 Features

* 🌌 Futuristic sci-fi inspired UI
* ⚡ Smooth hover and glow effects
* 🎮 Interactive gaming themed sections
* 📱 Fully responsive design
* 🛰️ Hero section with CTA buttons
* 🛡️ Fighter class showcase
* 💥 Tactical simulator section
* 💳 Battle pack pricing cards
* 📰 Gaming press testimonials
* ❓ FAQ section
* 📧 Beta enlistment/contact form
